# Penjelasan Sedikit
Sekilas, program tersebut adalah game pilihan cerita (text adventure) bertema zombie dengan fitur:

Menu utama:

Mulai Game

Daftar Ending

Keluar


Pemain memasukkan nama karakter.

Cerita bercabang menggunakan if, elif, dan else.

Terdapat beberapa ending:

Good End (Safe In Chaos)

Bad End (Exposed)

Bad End (Die)


Ending yang ditemukan disimpan ke file ending.txt dan bisa dilihat kembali melalui menu "Daftar Ending".
